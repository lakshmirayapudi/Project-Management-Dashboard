# Project Management Dashboard

## Overview
Project Management Dashboard is a React JS frontend project designed to manage projects and tasks. It provides a simple dashboard interface to view total projects, completed tasks, pending tasks, and task suggestions.

## Features
- Dashboard statistics cards
- Ongoing projects page
- Completed tasks page
- Pending tasks page
- Add new task functionality
- Search suggestion dropdown
- Responsive and clean user interface

## Technologies Used
- React JS
- JavaScript
- HTML
- CSS
